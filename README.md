# 🛍️ Customer Behaviour Analysis | End-to-End Data Analytics Project

An end-to-end retail data analytics project that transforms raw customer shopping data into actionable business insights using **Python (Pandas), PostgreSQL, SQL, and Power BI**.

---

## 📌 Project Overview

This project simulates a real-world data analytics workflow by:

- Cleaning and preprocessing raw retail data using **Python (Pandas)**
- Performing Exploratory Data Analysis (EDA)
- Loading cleaned data into **PostgreSQL**
- Solving business problems using **SQL**
- Creating an interactive **Power BI Dashboard**
- Generating business recommendations based on data insights

## 🔍 Key Business Insights

- Analyzed **3,900 customer records**.
- Clothing generated the highest revenue and sales.
- Male customers generated significantly higher revenue than female customers.
- Loyal customers accounted for nearly **80%** of all customers.
- Only **27%** of customers had an active subscription.
- Subscribers and non-subscribers had almost identical average spending.
- Young Adults contributed the highest sales and revenue.
- Some products relied heavily on discounts to increase sales.
- Express Shipping customers spent slightly more than Standard Shipping customers.

---

## 💡 Business Recommendations

- Improve subscription benefits.
- Increase customer acquisition campaigns.
- Reduce dependency on heavy discounts.
- Focus marketing on high-performing categories.
- Launch personalized campaigns for female customers.
